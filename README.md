# 以声绘色：儿童绘本多模态共创平台StoryCanva🎉

在儿童教育领域，传统的教育模式往往侧重于知识的传授和技能的培养，而忽视了孩子们天马行空的想象力和创造力的培养。为了解决这一问题，儿童绘本多模态共创平台StoryCanva应运而生。该平台结合拼贴画、创意写作和生成式人工智能（AIGC）技术，旨在激发儿童的想象力和创造力，提供一个安全、有趣且富有启发性的故事创作环境。平台主要包含儿童对话模型、素材生成模型和拼贴插图生成模型，能够根据儿童的口头描述生成图像素材，并通过图像素材的拼贴与机器人共同完成故事插图的创作，实现人机绘本共创。

## ⭐部署算法

### 儿童对话模型

API封装目录：`./backend/Qiaoban`

在儿童对话模型上，平台选用了Qiaoban-7B进行对话模型的部署。QiaoBan是由哈尔滨工业大学情感计算组开发的中文儿童情感陪伴对话系统，灵感来源于中国传统益智玩具七巧板。该系统基于baichuan-7B大模型，通过指令微调训练，结合儿童心理学理论与高质量儿童对话数据，优化了儿童情感对话体验。

### 素材生成模型


在素材生成模型上，平台选用了太乙Diffusion进行拼贴素材的生成。在素材生成模型上，平台选用了太乙Diffusion进行拼贴素材的生成。能够理解儿童的中文口头描述，并将其转化为视觉图像。这一过程不仅为儿童提供了一个直观的创作工具，而且极大地激发了他们的想象力和创造力。儿童可以通过简单的语音指令，引导太乙Diffusion生成各种风格和主题的拼贴素材，从而在艺术创作中实现个性化表达。

### 拼贴插图生成模型

目录：`./collage-diffusion`

在拼贴插图生成模型上，平台选用了Collage Diffusion进行拼贴画的融合生成，作为儿童艺术创作的辅助工具。Collage Diffusion的引入极大地丰富了QiaoBan平台的艺术创作能力，提供了一种创新的图像生成方式，让儿童能够以一种直观和互动的方式探索和表达自己的创意。

## 🎨运行流程

### 环境配置
运行`pip install -r requirements.txt`

### 运行平台
后端运行`python backend\run.py`，打开`http://127.0.0.1:5000`即可访问平台

### 运行测试
项目包含后端测试文件 `./backend/test_app.py`。要运行测试，请在项目根目录下执行（确保已安装如 `pytest` 等测试框架）：

```bash
pytest backend/test_app.py
```
或者如果使用 unittest 框架：
```bash
python -m unittest backend.test_app
```
测试结果将显示在终端中。
