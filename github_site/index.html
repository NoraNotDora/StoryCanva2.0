StoryCanva 系统介绍

通过对话引导、图像生成与拼贴融合，支持儿童完成绘本共创

算法能力

系统采用儿童对话模型、素材生成模型与拼贴融合模型，形成从故事构思到插图生成的完整链路。

儿童对话模型

基于 Qiaoban-7B 实现故事对话引导，帮助儿童梳理角色、情节和表达逻辑。

素材生成模型

基于太乙 Diffusion 将文本描述生成图像素材，支持多主题拼贴创作。

拼贴融合模型

基于 Collage Diffusion 完成多素材融合，生成统一风格的绘本插图。

系统功能

绘本创作：拼贴画板 + 故事机器人交互

成就激励：按创作数量解锁奖牌与进度

社区分享：发布作品、浏览画廊、查看详情

个人中心：信息编辑、作品管理、互动查看

成就激励模块

按创作数量发放奖牌并显示成长进度，增强持续创作动力。

社区分享模块

个人信息管理模块
